# Healthcare Data Warehouse & BI Project

## 📌 Overview
This project demonstrates a complete Business Intelligence solution using a healthcare dataset. It includes data warehousing, OLAP cube development, Excel analysis, and Power BI dashboards.

## 🛠 Technologies Used
- SQL Server (Data Warehouse)
- SSAS (Multidimensional Cube)
- Microsoft Excel (OLAP Analysis)
- Power BI (Reporting & Dashboards)

## 📊 Features
- Star schema data warehouse
- SSAS cube with hierarchies
- OLAP operations (Roll-up, Drill-down, Slice, Dice, Pivot)
- Interactive Power BI reports
- Drill-down and drill-through analysis

## 📁 Project Structure
- `1_DataWarehouse` – SQL scripts and warehouse design
- `2_SSAS_Cube` – Cube project files
- `3_Excel_OLAP` – Excel OLAP analysis
- `4_PowerBI` – Power BI dashboards
- `5_Documentation` – Final report
